Integration mocking in Node.js

## Installation

```
yarn add -D unmock
```

or

```
npm install --save-dev unmock
```

## Usage

```bash
$ unmock --help
Usage: unmock unmock <command>

Options:
  -h, --help             output usage information

Commands:
  init
```

## init

```bash
$ unmock init --help
Usage: init
```
